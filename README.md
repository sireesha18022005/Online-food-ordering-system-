#Online Food Ordering System — Transformer-based Sentiment Analysis

This project analyzes multilingual customer feedback from food delivery platforms using transformer models such as **BERT**, **DistilBERT**, **IndicBERT**, and **MuRIL**.  
It achieves up to **96% accuracy**, outperforming traditional CNN–LSTM and BiLSTM models.

##  Features
- Multilingual feedback analysis (English, Hindi, Kannada)
- Real-time inference using **FastAPI**
- Model interpretability with **SHAP** and **LIME**
- Deployment-ready with **Streamlit**

##  Models Used
- BERT  
- DistilBERT  
- IndicBERT  
- MuRIL  

##  Tech Stack
- Python  
- PyTorch  
- Hugging Face Transformers  
- FastAPI / Streamlit  
- SHAP, LIME
